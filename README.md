# Typescript
Learning typescript

## How to install
 <ul>
  <li>npm init -y</li>
  <li>npm install --save-dev typescript</li>
 </ul>

 ## How to compile
 <ul>
  <li>tsc filename.ts</li>
  <li>node filename.js</li>
 </ul>

 ## OR 
  <ul>  
    <li>tsc filename.ts --watch</li>
   <li>node filename.js</li
 </ul>
